## 📚 Sobre o repositório

Este repositório reúne exercícios resolvidos na plataforma **BeeCrowd** com o objetivo de praticar:

- lógica de programação;
- resolução de problemas;
- estruturas condicionais e de repetição;
- vetores e matrizes;
- entrada e saída de dados em JavaScript;
- organização e evolução nos estudos.

Além de servir como prática, este espaço também funciona como um registro da minha trajetória de aprendizagem em programação.

---

## 🚀 Tecnologias utilizadas

- **JavaScript**
- **Node.js** *(para execução local, quando necessário)*
